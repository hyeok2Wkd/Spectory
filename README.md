### #Commit Message Rule

| Message  | 설명                                                         |
| -------- | ------------------------------------------------------------ |
| Feat     | 새로운 기능 구현, 파일 생성 등 작업                          |
| Fix      | 에러, 버그 등을 수정 - ex 에러 난 상태로 커밋해두고 다시 에러 고칠 때 |
| Refactor | 간단한 변수명 등 수정                                        |
| Docs     | README 등의 문서 수정                                        |
| Chores   | 프로젝트 구조 등 코딩 외의 변경                              |
